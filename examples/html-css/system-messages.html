Implementation Notes

CSS Classes Required:

  • .system-message - Base message styles
  • .system-message-warning - Warning variant styles
  • .system-message-notice - Notice/announcement styles
  • .system-message-info - Informational message styles
  • .system-message-icon - Icon container styles
  • .dismiss-btn - Dismissal button styles

Accessibility Features:

  • role="alert" for urgent warnings
  • role="status" for informational messages
  • aria-live="assertive" for critical alerts
  • aria-live="polite" for non-critical updates
  • aria-label on dismiss buttons for screen readers

Color Tokens Used:

  • --color-error-warning-orange - Warning icon background
  • --color-info-icon-blue - Info/notice icon background
  • Background colors use hex values for specific warning/notice styling
