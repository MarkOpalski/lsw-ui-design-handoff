Implementation Notes

CSS Classes Required:

  • .button-base - Base button styles
  • .btn-primary - Primary green button
  • .btn-secondary - Primary grey button
  • .btn-outline - Outline style buttons
  • .btn-loading - Loading state styles
  • .btn-small - Small variant button
  • .button-disabled - Disabled state

Color Tokens Used:

  • --color-success-green - Primary button background
  • --color-grey-primary-bg - Secondary button background
  • --color-grey-footer-light-bg - Outline button background
  • --color-base-white - Button text on colored backgrounds
  • --color-base-card-text - Button text on light backgrounds
